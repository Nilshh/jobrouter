# Job Scraper für IT-Führungspositionen

Eine automatisierte Webanwendung zur täglichen Suche nach Stellenanzeigen für CIO, CTO, Leiter IT, Direktor IT und Head of IT auf verschiedenen Jobbörsen.

## Features

✅ Automatisierte Suche auf LinkedIn, Stepstone, Xing und Jobware  
✅ Benutzerfreundliche Web-Oberfläche  
✅ Speicherung in Excel für weitere Verarbeitung  
✅ Fehlerbehandlung und Logging  
✅ Docker-Support  
✅ REST-API für Integration
